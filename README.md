# ViajandoImg

Projeto focado em processamento e organização de imagens para o canal **Inteligência Mil Grau (ET Labs)**.

## Descrição
Este repositório contém scripts e ferramentas para automação de fluxo de trabalho com imagens, visando otimizar a criação de conteúdo audiovisual.

## Estrutura do Projeto
- `index.html`: Página principal.
- `jogo.html`: Módulo de gamificação.
- `mapa.html`: Visualização geográfica.
- `receitas.html`: Guia de processos.
- `assets/`: Recursos estáticos (imagens, ícones).

## Como usar
- Os scripts principais estão localizados na raiz do projeto.
- Certifique-se de configurar suas credenciais conforme as instruções de segurança do projeto.
